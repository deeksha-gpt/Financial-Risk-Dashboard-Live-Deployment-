# Financial-Risk-Dashboard
Full-stack financial risk analytics platform built with Python.

Features: • Portfolio Value at Risk (VaR) • Stress testing • ML anomaly detection • Basel III risk reporting • Interactive risk dashboard

Tech Stack: Python, Streamlit, FastAPI, Plotly, Scikit-learn
